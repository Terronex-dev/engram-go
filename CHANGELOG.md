# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-27

### Added
- Initial release
- Core types: MemoryNode, Entity, Link, EngramHeader
- File I/O with integrity verification (SHA-256)
- MemoryTree for navigation and search
- Semantic search with cosine similarity
- Text search by content
- Streaming reader for large files
- Cross-SDK compatibility with TypeScript, Python, and Rust
